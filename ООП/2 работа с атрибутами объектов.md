
**`getattr(obj, name)`** — получить значение атрибута по имени (строкой).

```
getattr(user, "name")  # то же самое, что user.name
```

**`setattr(obj, name, value)`** — установить значение атрибута.

```
setattr(user, "age", 25)  # user.age = 25
```

**`delattr(obj, name)`** — удалить атрибут.

```
delattr(user, "age")  # del user.age
```
